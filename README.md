# plantuml-view-2

plantuml file viewer


following modules are required, you have to install them by pip3.

* PyQt5
* PyQt5.sip
* python-magic
* watchdog
* PyInstaller

exec make-exe.sh , then you will get plantuml-view-2 in dist.
